# Default packages cross-emerged into every image's sysroot.
# Lifted from cross-stage.sh ADDITIONAL_PACKAGES, trimmed to light
# essentials.  Per-board extras go in boards/<board>/target-packages.txt;
# a board can drop a default with a `-atom` line (e.g. `-app-misc/fastfetch`).
#
# Deliberately NOT here (boards opt in via their own list):
# - heavy toolchains: sys-devel/mold, dev-lang/go, dev-build/cmake,
#   dev-lang/rust
# - wireless: net-wireless/iw, net-wireless/wpa_supplicant
# - app-misc/screen: cross-emerge fails at fowners root:utmp (utmp user
#   missing from the cross-prefix passwd) — do not re-add here
# - dev-build/meson: PEP517 wheel build is fragile under cross-compilation
#   (host/target python paths confused) — do not re-add here

# Networking / remote access
net-misc/openssh
net-misc/ntp

# Logging
app-admin/metalog

# Initramfs / minimal userland
sys-apps/busybox

# Basic interactive tools
app-editors/vim

dev-vcs/git
